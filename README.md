# ipint
Show ip addresses

Installation
-
```
git clone https://github.com/JackScripter/ipint.git
sudo mv ipint/ipint /usr/bin/
sudo chmod +x /usr/bin/ipint
```

Usage
-
```
[root@localhost ~]# ipint
=============================================
INTERFACE        ADDRESS
=============================================
lo               127.0.0.1/8
ens33            192.168.152.129/24
ens37            192.168.1.10/24
vmnet1           172.16.0.254/24
vmnet2           172.16.1.254/24
vmnet8           172.16.78.1/24

Gateway is 192.168.152.2
```
